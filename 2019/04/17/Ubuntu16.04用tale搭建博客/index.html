Ubuntu16.04用tale搭建博客

四月 17, 2019
  • Ubuntu
  • 博客

运行环境准备：

​ 1.Java环境（版本1.8以上并且一定要配置环境变量）；

2.安装MySQL并建库

​ mysql> create database tale default character set utf8 collate utf8_general_ci;

​ 3.下载tale博客系统

​ 直接到 https://github.com/otale/tale/releases 下载 tale.tar.gz

​ 接着使用 FileZilla 将该压缩包先在本地解压然后再上传到服务器上 （这里我上传在/home/admin/blog/ 目录下）

​ 然后在右边tale处右键鼠标修改文件权限为777（FileZilla可直接这样修改文件权限）

​ 然后用远程工具 Putty 登入服务器

​ 进入到tale所在文件夹

​ cd /home/admin/blog/tale

​ 再执行

​ sh tool start

​ OK.这里tale系统这部分就完成了

​ 4.安装nginx并配置IP

​ 首先安装nginx

​ sudo apt-get install nginx

​ 然后配置IP

​ sudo apt-get install vim （安装vim工具，若已安装请忽略）

​ sudo vim /etc/nginx/sites-available/default

​ 修改配置文件如下：（其他不同的部分加 # 字符注释掉即可）

​ server {

​ listen 80 default_server;
​ listen [::]:80 default_server ipv6only=on;
​
​ server_name 你的服务器的公网IP或域名; // 没有网址就写localhost
​ location / {
​
​ proxy_pass http://0.0.0.0:9000;
​
​ }
​ }

​

​ 最后

​ 按esc

​ “shift” + “：”

​ 输入wq! +回车保存并退出

​

​ 配置完成重新启动nginx

​ sudo service nginx restart

​

测试

​ 直接在浏览器输入你的域名或者IP

​ 进入设置界面（这里我当初搭时候没有截图就不放效果了）

​ 输入昵称加密码注册一下就能进了

​ 最后的效果如图


​ 进入管理界面

​ 直接在浏览器输 ： 你的IP或域名+/admin 就能进了。

至此 tale 博客搭建完成！
